# 요청 사항
'docs/3. 자료/과거 쪽지시험 알파테스트 레포(quiz-app)'때 컨셉을 바탕으로, 앱 내에서 학년별로 쪽지시험(앱에서는 '확인 평가'라는 이름)을 볼 수 있도록 해야 한다. (저 quiz-app을 자세히 볼 필요는 없다. 단순히 AI가 PDF에서 문제를 얼마나 잘 뽑는지 고모에게 보여드리기 위해 만든 임시 사이트이다. 구조, 코드를 적극적으로 참고할 필요 없다.)

일단 구현 위치는, 기존 하단 바가 개수가 많아서 진단 탭에 학습 진단, 진로진단, 확인 평가 3개를 모아서 보여주면 되겠다.

가장 이상적인 시나리오는 관리자가 직접 PDF 파일을 올리면, 실시간으로 PDF가 스캔되고(텍스트 추출, OCR 등을 AI API 써서 자동으로), AI가 문제+정답+해설을 자동 생성해서, 관리자가 검수 후 통과된 문제만 바로 배포되어야 한다.

그런데 당장 내일 급하게 필요하다고 하셔서 우선은 docs/1. 프로젝트/산청 우정학사 중등부 베타테스트/쪽지시험(확인평가) 기능 추가/교재 원본(중1, 중2, 중3) 여기에 있는 PDF를 로컬에서 ai agent 동원에서 내가 직접 문제를 추출한 후, 수동으로 내가 DB에 넣으려고 한다.
