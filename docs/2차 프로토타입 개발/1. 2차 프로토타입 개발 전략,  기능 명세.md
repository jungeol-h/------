# 2차 프로토타입 개발 전략 및 기능 명세

## 개요
1차 프로토타입에 대한 클라이언트 검토 피드백을 반영한 2차 프로토타입이다.
1차와 동일하게 시연 목적의 프론트엔드 프로토타입으로, 백엔드 없이 Mock 데이터 + LocalStorage로 동작한다.

## 앱 명칭 변경
| 항목    | 1차                     | 2차                  |
| ----- | ---------------------- | ------------------- |
| 앱 타이틀 | 안동형 자기주도학습·진로성장 관리 시스템 | **나르샤** (가칭)        |
| 부제    | —                      | 안동시 자기주도 학습 센터 NAVI |
| 기관명   | —                      | 안동시 장학회             |

코드 내 "안동형 자기주도학습 관리 시스템" 텍스트 전체를 교체한다.

## Mock 데이터 변경사항

### 마인드 레코드 스키마 변경
```js
// 기존
{ id, studentId, date, emotion: '좋음'|'보통'|'힘듦', motivation: 1~5, confidence: 1~5, memo }

// 변경
{ id, studentId, date, confidence: -5~5, motivation: -5~5, mood: -5~5, memo }
// 알림 트리거: (confidence + motivation + mood) <= -6 || min(confidence, motivation, mood) <= -4
```

### 3줄 일기 레코드 추가
```js
{ id, studentId, date, praise: string, reflection: string, resolution: string }
```

### TODO 리스트 레코드 추가
```js
{ id, studentId, date, subject: string, plannedMin: number, done: boolean }
```

### 센터 이용시간 레코드 추가
```js
{ id, studentId, date, centerMinutes: number }  // 실제 방문 학습 시간(분)
```

### 상담/코칭/컴플레인 레코드 추가
```js
// 상담
{ id, studentId, educatorId, date, type: 'counseling', content, tags: [] }
// 컴플레인
{ id, studentId, date, content, status: 'open'|'closed' }
// 만족도
{ id, studentId, date, score: 1~5, comment }
```

## 시연 시나리오 (2차)
1. 학생 로그인 → 홈 탭 (날씨 마인드 이미지, 오늘의 학습 막대그래프, 미완료 과제 붉은 경고)
2. 학습 탭 → TODO 리스트 작성 → 타이머 학습 → 파이 차트 갱신
3. 마인드 체크 → 자신감/학습 의욕/기분 -4점 이하 입력 → 저장
4. 3줄 일기 탭으로 전환 → 칭찬/반성/다짐 입력
5. 매니저 전환 → 홈 탭 알림 확인 → 클릭 → 코칭 코멘트 작성 → 알림 해제
6. 매니저 홈 → 담당 학생 현황 (학습 총 시간 / 과제 미수행 / 상담 회차) 확인
7. 관리자 전환 → 홈 탭 (인원 현황, 상담/코칭/컴플레인 현황) 확인
8. 통계 탭 → 학습량/과제 이행도/마인드 총점 차트, 월간 센터 이용시간 확인
9. 리포트 인쇄 버튼 → window.print()

## 개발 순서 (권장)
1. Mock 데이터 스키마 업데이트 (`mockData.js`, `DataContext.jsx`)
2. 앱 명칭 전체 교체 (index.html, LoginPage, Header, AdminHomeTab 등)
3. 마인드 탭 전면 개편 (점수 체계 변경 + 3줄 일기 서브 화면)
4. 학습 탭 업데이트 (과목 13종, TODO 리스트, 파이 차트)
5. 과제 탭 업데이트 (출제자, 마감 시간, 파일 업로드 더미)
6. 진로 탭 업데이트 (제목 변경, 보고서 진행률 8단계)
7. 학생 홈 탭 업데이트 (날씨 마인드, 가로 막대그래프, 미완료 경고)
8. 매니저 홈 탭 업데이트 (3항목 지표, 클릭 세부 확인)
9. 관리자 홈 탭 업데이트 (인원 현황, 신규 메뉴들, 학교 현황 제거)
10. 관리자 통계 탭 업데이트 (센터 이용시간, 불필요 차트 제거)
