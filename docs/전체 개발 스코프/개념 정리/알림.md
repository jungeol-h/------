# 알림 체계 (Notification)

## 개념
시스템에서 특정한 사건(Event)이나 조치가 필요한 상황이 발생했을 때, 사용자의 화면 밖으로 즉각적인 메세지를 능동적으로 전달하여 사용자의 행동(Action)을 유도하는 트리거 시스템이다.

## 알림의 역할과 특징
* **적시성(Timeliness):** 출결 미입력, 과제 마감 임박 등 정해진 기한 내에 사용자가 누락한 입력을 독려하여 데이터 기입 빈도를 높인다.
* **인지 제고:** 리포트가 최종 발행되거나 코치의 피드백이 등록되었을 때 등 사용자가 플랫폼에 들어올 명분과 시기를 알려 재접속률(Retention)을 높인다.
* **위험 감지 안내:** 장기 지각/결석 패턴이나 멘탈 하락폭이 감지된 학생에 대해 담당 교육자 및 관리자에게 경고 알림(Alert)을 띄워 빠른 대처를 돕는다.

## 적용 플랫폼 환경
- 1차적 기획은 앱 내 알림을 기반으로 하는 **App Push** 연동 체계로 확립한다.
- 향후 시스템 확장에 따라 카카오 알림톡(Kakao Biz), 단문 문자 메세지(SMS), 자동 이메일 알림 등으로 채널을 확장해 나아갈 수 있는 구조를 전제로 한다.
