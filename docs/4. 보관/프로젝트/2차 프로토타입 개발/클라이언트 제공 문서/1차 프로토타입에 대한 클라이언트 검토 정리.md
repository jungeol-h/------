# NAVI 센터 자기주도 학습 관리 시스템 검토 1차
(개발자 주석) '아래 물어보는 문장들은 클라이언트가 직접 물어본 문장이다.'
(개발자 주석) '기본적으로 2차 프로토타입도 1차 프로토타입과 마찬가지로 기능들이 시연만 되면 되지 내부 로직까지 완벽하게 구현될 필요가 없다. 따라서 클라이언트의 구현 가능 여부 질문들은 모두 더미 데이터, 조건부 애니메이션 등등으로 시연이 가능한지 묻는 것이다. 기본적으로 내가 모든 질문들을 검토했을 때 불가능한 구현은 없어보인다. 따라서 질문들도 추가 요청사항으로 동일하게 생각하면 되겠다.'
## 시스템 추가 설명 (기능 상세)
아래는 NAVI 센터 자기주도 학습 관리 시스템-“날다”에 대한 추가 설명이다.
(개발자 주석) '1차 때 다소 모소했던 기능들이 구체화되었다. 따라서 이 섹션 내용은 전체 개발 스코프 안에 개념 정리, 유저별 폴더에 최신화해야 한다. 어느정도 기존에 내가 정리한 내용이 겹치지만 기존 내 정리보다는 클라이언트 워딩과 의미가 더 반영되도록 전체 문서들을 업데이트 해야한다. Claude는 이 내용을 숙지하고 작업을 나 대신 수행해야 한다.'
### 1) 학생의 학습 상태 진단 기능
학습자의 현 상태를 6개 유형으로 분류하여 문제 상황을 구체적으로 규정하고 이에 맞는 학습 조언을 제공하여 변화과정을 관리할 수 있도록 한다.(매뉴얼 기반)
### 2) 학생의 관리 기능
#### ① 학습관리
국어, 영어, 수학, 과학, 사회, 도덕, 역사(한국사), 기술가정, 한문, 정보 등 학교 교과서 목록과 교양 독서, 진로 독서, 진로 탐구 의 비교과 활동을 관리 항목으로 한다.
##### <학습 관리 방법>
 1. 학습 타이머로 시간 기록 : 시작 시간과 종료시간 체크 기록
   -학습시간 누적 기록 데이터화
   -1일 학습 시간 4시간 이상 10점. 4시간 미만 3시간 30분 초과 9점, 3시간 초과 3시간 30분 미만 9점, 3시간 미만 2시간30분 초과 7점, 2시간 30분 미만  2시간 초과 5점, 2시간 미만 1시간 30분 초과 3점, 1시간 30분 미만 1시간 이상 1점. 이하 0점
 2. 학습 종료 후 학습량, 집중도, 만족도, 효율성 자가 평가(10점 만점)
* 가와 나를 합산하여 50점 만점을 백분위 점수로 환산하여 자기 주도 학습 지수로 산정
 3. 학습 계획표 작성 – 매일 다음날 학습 계획을 TO DO 리스트로 작성
 4. 학습 계획 실행도 점검 – 매일 TO DO 리스트를 작성하면서 그날 학습 계획 실행도를 점검
 5. 위 4개 항목의 결과를 점수화하여 자기주도 학습 
 ###### 궁금한 사항
 - 3.과 4.를 학생이 입력하면 1.과와 2.를 통해 자동으로 완료 상태가 데이터화 될 수 있는 지 궁금함. 그러니까, 다에서 작성한 TO DO 리스트 대로 가에서 교과를 클릭하고 계획한 시간 동안 학습을 하고 종료하면 TO DO 리스트에 완료한 것으로 자동으로 표시가 되는지...(과제 점검처럼)
 - 만약 3.과 4.의 기능 구현이 어렵다면 자기 체크리스트로 작성하고 스스로 점검할 수 있는 페이지로 구성은 가능한지... 또 자가 점검 결과를 데이터로 누적하고 리포트할 수 있는지..
#### ② 마인드 관리
1. 1. 자신감, 2. 학습 의욕, 3. 정서적 안정 이 세 가지 항목
2. 주 3회 자가 평가
3. 세 가지 항목 각각 –5점~+5점으로 점수화하여 평가
4. 주 별 리포트 정서 추이 그래프로
5. 매일 기록 사항, 칭찬일기 한 줄, 반성일기 한 줄, 다짐일기 한 줄.
요청사항: 마인드 자가 점검 페이지와 일기 페이지를 분리.
#### ③ 진로 탐구와 설계
1. 진로 적성 진단 – 커리어넷 3종 검사 결과와 자체 진로 진단 웹 활용
2. 진로탐구 과정 관리 – 진로 탐구 보고서 작성
3. 진로 진학 설계 – 1:1 진로 컨설팅 최대 3회
4. 다.의 결과를 진로 로드맵으로 데이터화(페이지에 진로 적성, 흥미, 목표 등으로 시각화)
#### ④ 과제관리
1. 코치가 과제 부여.
	1. 학생과 학부모에게 알림 기능
	2. 과제 완료 시점 지정과 제출 관리
	3. 수동으로 과제 확인 결과 기록 가능
#### ⑤ 성취도 관리
1. 학생 별로 평가 과목 지정(학습 컨설팅 후 결정)
2. 월 4회까지 주기적 평가(성적추이 기록)
3. 과목별 교과 학습 코칭과 연계
#### ⑥ 만족도 조사
학생에게 과제 부여 형태로 시행 가능한가? 불가능하면 기존대로 구글 설문지로 진행
## 화면별 피드백
### 메인 화면
1. 어플 제목은 “나르샤”(가칭)
2. 부제로 안동시 자기주도 학습 센터 NAVI
3. 하단 부서명 안동시 장학회
### 학생 홈 탭
1. 오늘의 마인드는 나의 학습 옆에 날씨처럼 이미지로 표시
2. 자기주도 학습지수와 같은 크기로 오늘의 학습을 과목과 시간이 표시되도록. 가로형 막대그래프
3. 과제 현황에 미완료 과제는 붉은 색으로 경고등표시
4. 하단에 메뉴에 3줄일기 페이지 추가
### 학생 학습 탭
1. 학습 타이머에 과목 선택 항목 : 국어, 영어, 수학, 과학, 사회, 도덕, 역사(한국사), 기술가정, 한문, 정보, 교양 독서, 진로 독서, 진로 탐구
2. (중요) 이 페이지에 TODO 리스트 작성 메뉴가 있어서 누르면 새로운 페이지가 열리고 기록가능하면 좋을 듯.(캘린더 기능과 연동이 어려운가?)
3. 학습 기록은 파이형 그래프 하나에 통합하면 좋을 듯.
### 학생 과제 탭
1. 과제 목록에 각 과제 제목 아래 과제 출제자(국희샘 등)이름
2. 마감 날짜와 시간을 옆에 별도의 단추로
3. (중요)과제 제출하기 버튼으로 과제 이미지나 파일을 업로드할 수는 없는지
4. (중요) 코치나 상담사가 과제 확인을 하고 그 결과를 학부모도 열람하거나 알림을 받을 수 있으면 좋을 듯.
### 학생 마인드 탭
1. 1. 자신감, 2. 학습 의욕, 3. 기분 이 세 가지 항목
2. 주 3회 자가 평가(날짜 기록란 필요)
3. 세 가지 항목 각각 –5점~+5점으로 점수화
4. 제목은 마인드 체크
5. 한마디(선택)을 “제 얘기 좀 들어주세요.”(선택)으로
6. 여기에 메모한 내용을 관리자에게 알림으로 전달할 수 있었으면 좋겠음
### 학생 진로 탭
#### 진로 검사 전 페이지
1. 제목을 진로설계로
2. 진로 활동 기록 아래 “진로 탐구 보고서 진행률”막대 그래프 첨부
3. 보고서 활동 진행 단계 : 주제선택 – 자료수집 – 개요작성 – 초고 쓰기 – 첨삭1 – 고쳐쓰기 – 첨삭2 – 완성
#### 진로 검사 페이지
1. 이 페이지는 [[진로 설계 검사 개발 명세서]] 참고.
### 학습 매니저 홈 탭
1. 상기한 학생 수정 사항이 반영됨이 따라 전반적으로 개선 필요
2. 담당학생 현황을 자기주도 학습지수 말고 학습 총 시간과 과제 미수행. 상담 누적 회차, 이 세 가지가 나오도록
3. (중요)단추로 만들어서 눌러서 각 항목의 세부 내용을 확인할 수 있으면 더 좋겠음
