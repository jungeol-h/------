# 🧠 01. 학습 진단 기본 구조 및 문항

이 문서는 학습 진단의 기본 분류 체계, 유형 산출 엔진 규칙, 그리고 실제 사용자에게 제공될 24개의 진단 문항을 정의합니다.

---

## 1. 진단 기본 체계 및 유형 정의

### 🔹 1.1 핵심 3유형 (기준 축)

| 영역 | 유형명 | 구성 요소 |
| :--- | :--- | :--- |
| **주도성** | **주도형** | 자아효능감 + 학습의지 + 비전 |
| **전략성** | **전략형** | 효율성 + 학습환경 |
| **성실성** | **노력형** | 성실성(노력형) |

### 🔹 1.2 유형 산출 규칙 (엔진)

1. **각 영역 점수 계산**: 5점 척도 문항의 평균값을 구함.
2. **A~E 변환 기준**:
  * 90점 이상: A
  * 80점 이상: B
  * 70점 이상: C
  * 60점 이상: D
  * 60점 미만: E
3. **핵심 유형 계산**:
  * **주도형** = (자아효능감 + 학습의지 + 비전) 평균
  * **전략형** = (효율성 + 학습환경) 평균
  * **노력형** = (성실성 자립 점수)
4. **최종 유형 조합 선택**:
  * 상위 2개 유형을 선택하여 조합 (예: 주도 + 전략 -> '주도적 전략형')
  * 단독 유형인 경우: 주도형, 전략형, 성실형(노력형)으로 표기
5. **부족 영역 판단 (접두어 생성)**:
  * 6개 영역 중 가장 점수가 낮은 영역을 기준으로 접두어 생성
  * **B (평균 80 미만)**: "~이 요구되는"
  * **C (평균 70 미만)**: "~이 필요한"
  * **D/E (평균 60 미만)**: "~이 시급한"

### 🔹 1.3 최종 유형 명칭 구조

👉 **[부족 영역 표현] + [핵심 유형 조합] + "학습자"**

* 예: "주도성이 요구되는 전략적 노력형 학습자"

## 2. 최종 진단 문항 (24문항)

**선택지 (Likert 5점 척도)**:
① 전혀 아니다 / ② 아니다 / ③ 보통이다 / ④ 그렇다 / ⑤ 매우 그렇다

### 🟦 1. 자아효능감 문항

1. 나는 공부를 하면 결과가 좋아질 수 있다고 믿는다
2. 나는 어려운 문제를 풀기 위해 여러 번 반복하는 편이다
3. 나는 시험 성적은 항상 내가 공부한 만큼의 결과라고 생각한다
4. 나는 어떤 과목이든 노력하면 잘할 수 있다고 믿는다
5. 나는 살면서 어려운 일이 있어도 결국 해낼 수 있다고 생각한다

### 🟩 2. 학습의지 문항

1. 나는 해야 할 공부를 미루지 않고 시작하려고 한다
2. 나는 공부가 하기 싫을 때도 이겨 내려고 노력한다
3. 나는 목표를 세우면 끝까지 해내려고 한다
4. 나는 모르거나 어려운 공부도 포기하지 않고 해내려고 노력한다
5. 나는 결과가 안 좋더라도 다음을 기대하고 노력한다

### 🟨 3. 효율성 문항 (학습 전략)

1. 나는 공부할 때 중요한 내용과 덜 중요한 내용을 구분한다
2. 나는 틀린 문제는 틀린 이유와 정답의 근거를 반드시 확인한다
3. 나는 수업 후 배운 내용을 요약 정리하고 복습한다
4. 나는 암기나 문제 풀이를 할 때 시간을 재면서 한다
5. 나는 나에게 맞는 학습방법을 찾기 위해 시도하고 활용한다

### 🟪 4. 노력형 문항 (성실성)

1. 나는 주어진 과제는 빠짐없이 모두 수행한다
2. 나는 매일 공부 시간을 정해 놓고 지키려고 한다
3. 나는 매일 공부할 내용을 정하고 실천하려고 한다
4. 나는 계획을 세우고 그대로 실천하는 편이다
5. 나는 매일 꾸준히 공부하는 습관이 있다

### 🟫 5. 학습환경 문항

1. 나는 공부할 때 방해되는 것(핸드폰 등)을 줄이려고 한다
2. 나는 집중이 잘 되는 장소에서 공부하려고 한다
3. 나는 공부 전에 주변을 정리하고 시작한다
4. 나는 공부할 때 다른 행동이나 생각을 줄이려고 한다
5. 나는 필요한 학습 도구(문구나 앱 등)를 잘 갖추고 있다

### 🟥 6. 비전 문항 (목표의식)

1. 나는 내가 왜 공부해야 하는지 알고 있다
2. 나는 앞으로 하고 싶은 일이나 목표가 있다
3. 나는 목표를 이루기 위해 지금 공부가 필요하다고 생각한다
4. 나는 구체적인 진로 목표와 진학 목표가 있다
5. 나는 진로 목표를 이루기 위해 지금 해야 할 공부를 계획한다
