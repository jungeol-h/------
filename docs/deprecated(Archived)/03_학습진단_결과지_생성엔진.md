# 🧠 03. 학습 진단 결과지 생성 엔진 및 리포트 디자인

이 문서는 진단 결과를 바탕으로 최종 리포트(웹/PDF)를 생성하는 종합 코칭 문장 조합 엔진과 시스템 자동 연결 파이프라인, 그리고 결과지의 시각적 디자인 구조를 정의합니다.

---

## 1. 종합 총평 생성 엔진 (엔진 로직)

### 🔹 1.1 총평 4단 구조
1. **현재 상태 진단**: 현재 수준 및 특징 설명
2. **강점 기반 가능성**: A/B 영역을 활용한 긍정적 전망
3. **보완 핵심 문제**: D/E/C 영역 중심의 문제 진단
4. **실행 방향 제시**: 구체 과제 3개 연결

### 🔹 1.2 기본 시스템 템플릿
> "현재 학습 상태는 **{전체 상태}** 단계에 있으며, 학습의 흐름이 **{특징 설명}** 형태로 나타나고 있습니다. 특히 **{강점 영역}**에서는 안정적인 기반을 갖추고 있어, 올바른 방향이 잡히면 성과로 빠르게 연결될 수 있는 상태입니다.
>
> 반면 **{보완 영역}**에서는 **{핵심 문제}**가 나타나고 있어 학습의 효율과 지속성이 제한될 가능성이 있습니다. 이 부분은 단순한 의지의 문제가 아니라, 학습 구조와 실행 방식의 개선이 필요한 단계입니다.
>
> 앞으로는 **{핵심 실행 방향}**을 중심으로 학습을 재구성해야 합니다. 특히 **{구체 과제1}, {구체 과제2}, {구체 과제3}**을 꾸준히 실천하면 학습의 흐름이 안정되고 성과로 이어질 가능성이 매우 높습니다."

### 🔹 1.3 통합 코칭 템플릿 (마인드 및 실천 구조)

웹에서 최종적으로 조합될 "마인드 코칭"과 "실천 코칭" 작성 엔진입니다.

🧠 **마인드 코칭 템플릿 (전제 → 해석 → 통합 결론 → 방향)**
> 지금 상태는 **{가장 낮은 보완 영역}**의 부족함으로 영향을 받고 있는 단계입니다. 특히 **{중간 보완 영역}**이 함께 작용하면서 전반적인 학습 흐름이 불안정해질 수 있습니다. 하지만 현재의 탁월한 강점인 **{강점 영역}**을 기반으로 충분히 빠르고 쉽게 회복과 성장이 가능한 상태입니다. 앞으로는 단기적인 결과보다 과정과 기준을 중심으로 학습을 점검하고 스스로를 바라보는 태도를 만들어야 합니다.

🔧 **실천 코칭 템플릿 (전제 → 구체 행동 → 반복 구조 → 실행 결론)**
> 지금 시점에서는 너무 많은 것을 한 번에 바꾸려 하기보다, 가장 핵심적이고 구체적인 한 가지 행동부터 안정시키는 것이 중요합니다. 특히 **{핵심 보완 영역의 실천 과제}**를 먼저 단단히 실천하시면 전체 학습 흐름이 긍정적으로 달라질 수 있습니다. 이 과정을 반복하면서 추가적으로 **{추가 보완 영역의 실천 과제}**까지 연결한다면 학습의 안정성이 크게 높아집니다. 앞으로는 하루 단위의 아주 작은 실천을 기준으로 꾸준히 반복하는 것이 가장 핵심적인 전략입니다.

### 🔹 1.4 실제 합성을 통한 출력 예시 (완성형)

* **테스트 사례**: 자아효능감 C / 학습의지 B / 효율성 A / 노력형 B / 학습환경 A / 비전 C
* **최종 산출 유형**: 주도성이 요구되는 전략적 노력형 학습자
* **종합 진단 (생성 결과)**:
  `현재 학습 상태는 전반적으로 성장 가능성이 높은 단계에 있으며, 학습의 부분적인 흐름이 안정되어 있으나 약간의 일관성이 부족한 형태로 나타나고 있습니다. 특히 효율성과 학습환경에서는 안정적인 기반을 잘 갖추고 있어, 올바른 방향이 세워지면 학습 성과로 자연스레 연결될 수 있습니다. 반면 자아효능감과 비전에서는 본인의 성취 방향성과 자기 확신이 충실히 구축되지 않아 학습 지속성이 흔들릴 가능성이 높습니다. 따라서 단순히 의지를 채우려 하기보다, 본인의 학습 과정을 점검하고 작은 성공 경험을 통한 목표 구조화를 진행해야 합니다. 특히 평소에 요점정리 15분을 배분하고, 꾸준히 오답노트 분석 루틴을 만들면 전체 방향성과 성취도가 같이 향상될 것입니다.`

## 2. 리포트 출력 가이드 및 디자인

### 🔹 2.1 페이지 구성 (웹/PDF)

* **1페이지**: 나의 학습 유형 결과, 레이더 차트, 한 줄 요약
* **2페이지**: 강점/보완점 상세 분석, 6개 영역별 간단 코멘트
* **3페이지**: 나를 바꾸는 통합 코칭 (마인드 + 실천 체크리스트)

### 🔹 2.2 디자인 레이아웃 (인포그래픽)

1. **중앙**: 최종 유형 명칭 (가장 크게 강조)
2. **좌측**: 6개 영역 레이더 차트 및 한 줄 해석
3. **우측**: 강점 영역(🟢) 및 보완 영역(🟠) 핵심 키워드
4. **하단**: 마인드 코칭 메시지 및 실천 체크박스 카드 3장

---

## 3. 시스템 자동 연결 로직 요약 (파이프라인)

1. `Score(24문항)` -> `Avg(6영역)` -> `Grade(A~E)`
2. `Sum(주도/전략/성실)` -> `Top 2 types` -> `Type Name`
3. `Min Grade` -> `Prefix` -> `Final Type Result`
4. `Min Area` -> `Template Mapping` -> `Coaching Text`
5. `Low Areas` -> `Task Selection` -> `3 Week Cards`
